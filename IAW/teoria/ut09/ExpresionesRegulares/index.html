Ejercicio991001ExpresionesRegulares.

Enunciado :

Hacer un Web que muestre 5 o más expresiones regulares y las pruebe frente a distintas cadenas, mostrando el resultado. Por cada expresión regular : Escribir el objetivo. Escribir un párrafo bajo ella que la explique. Dejar una casilla para meter una cadena y un botón que indique si la cadena es válida frente a esa expresión regular o no. Como sugerencia, estaría muy bien si cambiase de color de verde a rojo, por ejemplo. Otra sugerencia es que se puede plantear con el paso de parámetros al script o usar JavaScript o ambos.
